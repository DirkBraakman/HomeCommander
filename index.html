Development Journey

This project took me through 24 development steps over 2 months, from initial concept to final product.

Research & Planning

Created mood boards, explored hardware platforms, and chose Raspberry Pi + Electron as the foundation.

Learning Web Technologies

Studied HTML, CSS, and JavaScript to build the user interface. Created my first Electron application.

Home Assistant Integration

Implemented REST API communication with Home Assistant. Built device discovery and control logic.

Hardware Assembly

Wired rotary encoders, buttons, and touchscreen to Raspberry Pi. Created a functional prototype.

GPIO Integration

Overcame compatibility issues between Node.js, Electron, and GPIO libraries by downgrading to Pi OS Bullseye.

Final Polish

Refined the interface, cleaned up code, and created the final HomeCommander v3.1 with full functionality.
